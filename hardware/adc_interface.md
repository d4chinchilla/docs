# ADC Interface

The ADC board has the following wires:

| Wire | Colour | Purpose |
|------|--------|---------|
| 1.   | Red    | 5v0     |
| 2.   | Brown. | 3v3.    |
| 3.   | Black. | GND.    |
| 4.   | Blue.  | D1.     |
| 5.   | Blue.  | D2.     |
| 6.   | Blue.  | D3.     |
| 7.   | Brown. | A4.     |
| 8.   | White. | CLK.    |
| 9.   | Blue.  | D4.     |
| 10.  | Yellow.| ~CS.    |
| 11.  | Brown. | A3.     |
| 12.  | Brown. | A2.     |
| 13.  | Brown. | A1.     |
